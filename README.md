#Fonkin-Weather

Learning React Native by building fokin weather APP

Axios 사용하여 날씨 API 정보를 가져와서 

앱 사용자의 경도 위도를 통하여 Location파악하여

해당 지역의 날씨, 온도를 표시해주었습니다.

날씨에 따라 UI가 바뀌도록 하였습니다.
